Возможности системы
Модуль "Проверка"
  • Формирование шаблонов проверок (каталогов контрольных вопросов)
  • Детализация требований к свидетельствам и пояснениям в разрезе контрольных вопросов
  • Планирование комплексных проверок
  • Формирование реестра комплексных проверок
  • Формирование программ оценки уровня «зрелости» процессов
  • Распределение контрольных вопросов по исполнителям, верификаторам и аудиторам
  • Проведение проверок, включая процедуры самооценки, верификации, валидации (оценки уровня «зрелости»)
  • Агрегация оценок
  • Учет на всех этапах проверки соответствующих комментариев и пояснений к качественным и количественным оценкам
Модуль "Контрольные мероприятия"
  • Регистрация, редактирование и контроль недостатков, выявленных в ходе оценки уровня «зрелости» процессов
  • Формирование мероприятий по устранению выявленных недостатков
  • Обеспечение мониторинга контрольных мероприятий
  • Возможность выявления недостатков и рисковых событий в рамках целевых проверок
Модуль "Отчеты"
  • Итоговые отчеты по комплексным проверкам
  • Аналитические отчеты по всем стадиям проверки (программа проверки, отчет о самооценке, задание на оценку по существу, отчет по итогам оценки по существу)
  • Отчеты о ходе мероприятий, направленных на устранение выявленных отклонений по результатам проверок

Потребность в централизованном инструменте управления ИТ между всеми субъектами.
